Boost前言

Boost作为准标准库，提供了比标准库还丰富的工具和组件，在那个C++标准库还没有统一不同平台下的线程、日期、同步机制的C++03甚至更早，Boost就将他们统一了起来，伟大之处不用多说，更重要的是，Boost源码中相较于C++标准库的源代码，可读性大大增强，对学习编程者的思想很有帮助，对我学习泛型编程和了解标准库的实现提供了一条可行的路径。

Boost的编译

编译器：VC++ 2017 15.9.1 1916
源码地址 : https://dl.bintray.com/boostorg/release/1.68.0/source/boost_1_68_0.zip

所有命令 :

cd boost_1_68_0 // 进入代码根目录
.\bootstrap.bat // 编译编译工具b2/bjam，这两个是一样的东西
bjam stage --toolset=msvc-14.1 link=static runtime-link=static threading=multi debug release cxxflags="/Zc:threadSafeInit- " // 编译命令

参数说明：

architecture是指定处理器架构，arm/mips等，这里不用指定。参考：https://www.boost.org/doc/libs/1_55_0/doc/html/bbv2/reference.html
cxxflags="/Zc:threadSafeInit- "这个一定要加，是为了兼容xp上运行

编译指定库：

bjam --with-atomic stage --toolset=msvc-14.1 address-model=32 link=static runtime-link=static threading=multi debug release cxxflags="/Zc:threadSafeInit- "

./b2 --show-libraries
